# Sistema de cadastro declientes

Este projeto é uma sistema simples para cadastro de cliente.
O obejetivo é registrar infotmções basica como nome, email e telefone.

Este repositorio esta sendo usado para aprender os conceitos de Git, como versionamento, comits e envio para o repositorio remoto.

Este projeto está sendo desenvolvido em aula.
Novas funcionalidades serão adicionadas.

Testando novas mudanças atráves de ide da visual code.
